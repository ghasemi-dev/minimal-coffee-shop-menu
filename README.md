# کافی‌شاپ · منوی هوشمند

یک صفحه منوی کافی‌شاپ تعاملی و واکنش‌گرا که با HTML، CSS و JavaScript خالص ساخته شده است.  
کاربران می‌توانند محصولات را بر اساس دسته‌بندی فیلتر کنند، در منو جستجو نمایند و وضعیت فیلترها به‌صورت خودکار در مرورگر ذخیره می‌شود.

## ویژگی‌ها
- نمایش لیست محصولات به همراه تصویر، توضیحات و قیمت
- فیلتر بر اساس دسته‌بندی: نوشیدنی گرم، نوشیدنی سرد، ساندویچ، دسر و کیک، شیک و بستنی
- جستجوی لحظه‌ای در نام و توضیحات محصولات
- ذخیره‌سازی وضعیت فیلترها و جستجو در localStorage مرورگر
- طراحی واکنش‌گرا (Responsive) برای نمایش در انواع دستگاه‌ها
- دسترسی‌پذیری (Accessibility) با پشتیبانی از کیبورد و ARIA
